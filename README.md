# ICC T20 World Cup 2026 Dashboard

A modern, responsive analytics dashboard for ICC Men's T20 World Cup 2026.

## Features
- Team-wise squad view
- Player cards with role-based stats
- Interactive performance charts
- Global player search
- Clean, modern UI with dark analytics overlay

## Tech Stack
- React
- Vite
- Tailwind CSS + DaisyUI
- Recharts
